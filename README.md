# Genotyping Pipeline Tutorial (IDAT → GTC → VCF → PLINK)

Notebook praktikum bash-only untuk memahami pipeline genotyping microarray:
- IDAT → GTC (iaap-cli)
- GTC → VCF (bcftools +gtc2vcf)
- VCF → PLINK (plink)

## Cara pakai
1. Buka `genotyping_pipeline_bash_tutorial.ipynb` di JupyterLab
2. Edit `params.sh` (di dalam notebook) sesuai path file di komputer kamu
3. Jalankan cell berurutan

## Catatan
Data IDAT, reference FASTA, dan file besar lainnya tidak disertakan di repo.
